# Accidental_Drug_Related_Deaths_Case_Analysis Using Machine Learning and Anomaly Detection

# Overview
This project analyzes accidental drug-related death records to identify high-risk demographic groups, uncover hidden patterns, and detect unusual cases through machine learning techniques. The project combines clustering, classification, and anomaly detection methods to provide a comprehensive understanding of drug-related mortality trends.

# Dataset
The dataset contains demographic, geographic, and drug-related information associated with accidental drug overdose deaths.

# Objectives

- Identify patterns among overdose cases
- Discover high-risk population segments
- Predict overdose-related outcomes
- Detect unusual or anomalous cases

# Methodology
## Clustering Analysis
- K-Means Clustering
## Classification Models
- Random Forest
- XGBoost
## Anomaly Detection Models
- Isolation Forest
- Local Outlier Factor (LOF)
